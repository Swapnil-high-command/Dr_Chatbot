# Dr_Chatbot

 # 🤖 MEDI-BOT (AI-Based Medical Chatbot)

MEDI-BOT is a web-based medical chatbot designed to provide basic health-related suggestions based on user input. It allows users to interact through a chat interface and receive instant responses for common symptoms such as fever, headache, cold, etc.

This project demonstrates the integration of frontend technologies with a Python backend using Flask to create a real-time interactive application.

---

## 📌 Overview

The main goal of MEDI-BOT is to simulate a simple healthcare assistant that can guide users with preliminary advice. It is not intended to replace professional medical consultation but serves as a learning project for building chatbot systems.

---

## 🚀 Features

- 💬 Interactive chatbot interface  
- ⚡ Real-time communication using Flask API  
- 🏥 Basic medical suggestions based on symptoms  
- 🎨 Responsive and user-friendly UI  
- 🔄 Dynamic message display (user & bot)  
- 🧠 Rule-based response system  
- 🛠️ Easy to expand with AI integration
